## Run Locally

Clone the project

```bash
  git clone https://github.com/tareqnmd/next-basic.git
```

Go to the project directory

```bash
  cd next-basic
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
